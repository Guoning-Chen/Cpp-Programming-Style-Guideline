注意事项¶

  1. 由于水平有限，文中部分名词和句子翻译的似乎不太通顺，但未找到更好的译法，已在文中用下划线标出；

  2. 部分单词未找到合适的译法，在文中保留了英文，同样用下划线标出；

  3. 部分标记了下划线的英文单词在原文中出现多次，我推测可能是术语，在译者附录中给出了我自己的理解；

  4. 如果你发现翻译内容有误或对下划线部分有新的见解，欢迎在issue中指出，同样欢迎任何其他的意见。
